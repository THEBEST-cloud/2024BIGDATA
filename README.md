### Demo 说明
我们提供了在给定的训练集上一个训练 iTransformer 的示例代码，通过 `temp_train.sh` 调用 `train.py` 进行训练。 该训练代码提供给选手参考，选手可以根据自己的需求进行修改。

同时给出了 `index.py` ，该文件用训练好的模型进行推理并输出符合要求的文件。

注意，需要提交的只有推理代码和训练好的模型，训练代码暂时不需要提交。
